# Sistema buscador de jugadores Paddel

Node.js, Express, Passport, Mongoose, EJS y otros.



### Instalar

```sh
$ npm install
```

```sh
$ nodemon app.js

# http://localhost:5000
```

### MongoDB

"config/keys.js" para indicar tu mongoURI

### .ENV FILE


```sh
Archivo .ENV debe incluir:
    EMAIL_USER=tuemail@email.com
    EMAIL_PASS=tupassword

```
